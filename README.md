### Pre-commit Guide
가상환경 생성 후, 터미널에서 다음을 실행하면 pre-commit 설정 가능합니다.

    pip install -r requirements.txt (requirements.txt 위치 주의)
    npm install
    npm run prepare (team7-server로 이동하여 실행해야 함)
